# Структура иконок приложения

Эта папка содержит все иконки, используемые в приложении HealApp.

## Структура директорий

- `common/` - Общие иконки приложения (логотип, профиль, настройки и т.д.)
- `auth/` - Иконки для экранов аутентификации
- `navigation/` - Иконки для навигации (табы, меню и т.д.)
- `categories/` - Иконки категорий (роли, дневники, маркет и т.д.)
- `actions/` - Иконки действий (добавить, удалить, редактировать и т.д.)

## Форматы

Рекомендуется использовать:
- **SVG** для векторных иконок (лучшее качество, масштабируемость)
- **PNG** для растровых иконок (с разными разрешениями: 1x, 2x, 3x)

## Разрешения для PNG

- `icon.png` - базовое разрешение (24x24)
- `icon@2x.png` - для экранов с высокой плотностью (48x48)
- `icon@3x.png` - для экранов с очень высокой плотностью (72x72)

## Использование

Все иконки должны быть объявлены в `pubspec.yaml` в секции `assets`.

Пример использования в коде:
```dart
Image.asset('assets/icons/common/logo.png')
```
